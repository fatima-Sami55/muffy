# Spark 🌟 - A Sweet Journey into the World of Desserts 🍰🍩🍪

Welcome to **Spark**, a frontend web project I created to explore and practice responsive web design with a dessert-themed concept. 🌟

🔗 **Live Demo**: [spark-website.vercel.app](https://spark-website.vercel.app/)

---

## About Spark ✨

Spark is a minimalist dessert-selling website I developed using **HTML**, **CSS**, and **JavaScript**. While it's not a real e-commerce platform, it's a project built to enhance my skills in layout design, responsiveness, and basic interactivity.

---

## Features 🍫

- **Elegant Design**: Clean and user-friendly interface to simulate a dessert store experience.
- **Responsive Layout**: Fully adapts across desktops, tablets, and mobile devices.
- **Interactive Elements**: Includes basic JavaScript interactions and smooth animations.
- **Product Showcase**: Displays a variety of sample desserts.
- **Contact Section**: Static form for design and layout practice.
- **Simple Stack**: Built using HTML, CSS, Bootstrap, Bulma, and a little JavaScript.

---

## Specifications 🛠️

- **Languages Used**:  
  - **HTML** for structure  
  - **CSS** for styling  
  - **JavaScript** for interactivity  
  - **Frameworks Used**: Bootstrap and Bulma
- **Cross-Browser Compatibility**: Compatible with all major browsers  
- **Performance**: Designed for fast loading and smooth navigation
